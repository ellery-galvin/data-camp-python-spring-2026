# data-camp-python-spring-2026
Materials for an introduction to the Python language for data processing and scientific computing.

To interact with this code in your browser
- If you have not already, create a Github profile
- Click the "Fork" button at the upper right.  This will copy my repository into your own collection of repos.
- Accept the default settings and create the fork.
- Go to your profile, then your repositories, and open the new fork.
- Click the "Code" button in the upper right
- Click "Codespaces"
- Click "Create codespace on main"

A new tab will open with a cloud based environment for you to work in.  It will take several minutes for it to finish setting itself up.  When it's done
- Open "00_verify_setup.ipynb".
- It will take several more minutes for it to install packages.  You might see an error about the interpreter path, but you can safely ignore this.
- In the main area, there should be a cell with a play button in the upper left.  Click it.
- It may take several seconds, but you should python print a whole bunch of package names and versions.  It will also create a simple plot.  If you see all that, you're good to go!
